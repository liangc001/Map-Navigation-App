## 安装

### 系统要求

- Python 3.8 或更高版本
- 支持的操作系统：Windows 10/11, macOS, Linux

### 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/liangc001/Map-Navigation-App.git
   cd Map-Navigation-App
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 或使用 Make：
   ```bash
   make install
   ```

## 启动应用

```bash
python map.py
```

或使用 Make：

```bash
make run
```

## 注册与登录

### 首次使用

1. 点击「注册」按钮
2. 输入用户名和密码
3. 确认注册

### 登录

1. 输入已注册的用户名
2. 输入密码
3. 点击「登录」

### 注意事项

- 用户名不能重复
- 密码建议设置 6 位以上
- 数据存储在本地数据库

## 使用地图

### 界面说明

- **左侧边栏**：功能菜单
- **中央区域**：地图显示
- **底部/顶部**：状态信息

### 基本操作

- **缩放**：使用鼠标滚轮
- **移动**：拖拽地图
- **选择地点**：点击地图上的标记

### 可用地点

应用中预置了一些地点，您可以通过搜索快速找到。

## 路径规划

### 规划步骤

1. 点击「路径规划」菜单
2. 选择或输入起点
3. 选择或输入终点
4. 点击「计算」按钮
5. 查看规划结果

### 结果说明

- **总距离**：起点到终点的总距离
- **路径**：经过的地点序列
- **预计时间**：基于距离的估算

### 算法选择

- **Dijkstra**：单源最短路径
- **Floyd**：全源最短路径（适合多次查询）

## 常见问题

### Q: 应用无法启动？

A: 请检查：
- Python 版本是否 >= 3.8
- PyQt5 是否正确安装
- picture 目录是否存在

### Q: 地图显示异常？

A: 请确认：
- picture 目录中有背景图片
- 图片格式正确（JPG/PNG）

### Q: 忘记密码怎么办？

A: 当前版本密码存储在本地数据库，可以：
- 删除 `user_mes.db` 重新注册
- 或联系管理员重置

### Q: 如何添加新地点？

A: 当前版本需要在代码中添加：
1. 编辑 `dis33.py`
2. 添加新节点
3. 添加连接边
4. 重启应用

### Q: 路径计算很慢？

A:
- 地点数量过多时考虑简化图结构
- 使用 Dijkstra 代替 Floyd 算法

## 故障排除

### 数据库错误

如果遇到数据库错误：

```bash
# 删除数据库重新初始化
rm user_mes.db
# 然后重新启动应用
```

### UI 显示问题

尝试调整分辨率或缩放设置。
